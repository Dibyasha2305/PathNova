#  PathNova 
###  Your Personal Career Intelligence System

> Transform your resume, analyze your skills, and plan your career — all in one powerful Java-based system.

##  Problem

Many students face a common struggle:
-  Weak resume statements  
-  No clarity on required skills  
-  Confusion about career roadmap  
-  Difficulty in networking  

 There is a gap between **learning** and **industry readiness**


##  Solution

**PathNova Pro** is a modular Java-based system that helps users:

- Improve resume quality  
- Analyze job compatibility  
- Identify skill gaps  
- Generate professional messages  
- Plan structured career growth  


##  Features

###  Resume Enhancer
Enhances basic statements into **professional resume lines** using smart templates.


###  Resume Scorer
Evaluates resume quality based on:
- Action words  
- Technical skills  
- Description depth  
- Impact  


###  Job Matcher (Core Feature )
Compares resume with job description and provides:
- Match percentage  
- Skill category analysis  
- Missing skills  
- Smart suggestions  


###  Skill Gap Analyzer
- Calculates match level  
- Identifies missing skills  
- Suggests improvements  


###  Career Roadmap Generator
Provides structured roadmap:
- Beginner → Intermediate → Advanced  
- Role-based guidance  


###  DM Generator
Generates personalized cold messages for:
- Networking  
- LinkedIn outreach  


###  Smart File Management
Stores outputs in categorized files:
-resume.txt
-dm.txt
-analysis.txt
-roadmap.txt
-jobmatch.txt
